  • Was sind Backups?
    • Als Backup bezeichnet man eine Sicherheitskopie, die Daten im Fall eines Systemausfalls oder eines Datenverlustes zurückkopiert oder durch die sie wiederhergestellt werden können. Die Kopie kann auf einer externe Festplatte, der Festplatte des Computers oder auch auf einem USB-Stick erstellt werden. Auch online können die Daten gesichert werden.
  • Warum sollte man Backups machen?
    • Wenn Ihr Smartphone mal kaputt ist oder verloren geht, sind mit einem Backup die Daten wie Kontakte, Bilder und Videos noch verfügbar. Das Gleiche gilt für Ihren PC, wenn Sie Backups im Online-Speicher anlegen lassen. Sie können alle Daten wie Fotos und Dokumente später aus einem Backup wiederherstellen. Das ist zum Beispiel praktisch, wenn Ihr Smartphone kaputt ist und Sie es reparieren lassen müssen. Dabei wird das Gerät in der Regel komplett zurückgesetzt und alle Daten gelöscht. Spielen Sie dann das Backup ein, sind die Dateien wieder vorhanden.

Gibt es einen Weg ein Script automatisch wiederholt auszuführen? Also bspw. einmal pro Tag um 2 Uhr? Kannst du dir erklären wie das automatische Backups ermöglicht? Ja, es gibt verschiedene Möglichkeiten, ein Script automatisch zu wiederholen, zum Beispiel einmal pro Tag um 2 Uhr. Eine gängige Methode dafür ist die Verwendung von Cron-Jobs, die in Unix-basierten Systemen wie Linux und macOS verfügbar sind. Mit Cron kannst du Zeitpläne für die Ausführung von Scripts oder Befehlen festlegen. Um ein Script täglich um 2 Uhr auszuführen, würdest du einen Cron-Job wie folgt einrichten:

  1. Öffne das Terminal.
  2. Gib den Befehl: crontab -e ein, um die Cron-Tabellen zu bearbeiten.
  3. Füge eine Zeile hinzu, die so aussieht: 0 2 * /pfad/zum/deinem/script.sh

Das Pre-Element!

Das HTML-Element repräsentiert vorformatierten Text, der genau so dargestellt werden soll, wie er in der HTML-Datei geschrieben wurde. Der Text wird typischerweise mit einer nicht-proportionalen oder monospaced-Schriftart angezeigt. Leerzeichen innerhalb dieses Elements werden wie geschrieben angezeigt, mit einer Ausnahme. Wenn ein oder mehrere führende Zeilenumbrüche unmittelbar nach dem öffnenden Pre-Tag enthalten sind, wird das erste Zeilenumbruchzeichen entfernt.
